## Dependencies

The project uses the following key packages:

::card-group
:::card{icon="i-lucide-layout-template" title="Vue 3"}
Progressive JavaScript framework for building user interfaces.

[Documentation](https://vuejs.org/){target="\_blank"}
:::

:::card{icon="i-lucide-route" title="Vue Router"}
Official router for Vue.js — handles client-side navigation.

[Documentation](https://router.vuejs.org/){target="\_blank"}
:::

:::card{icon="i-lucide-database" title="Pinia"}
Intuitive, type-safe state management for Vue.

[Documentation](https://pinia.vuejs.org/){target="\_blank"}
:::

:::card{icon="i-lucide-move" title="Motion V"}
Animation library for smooth, performant UI transitions in Vue.

[Documentation](https://motion.dev/){target="\_blank"}
:::

:::card{icon="i-lucide-gallery-horizontal-end" title="Swiper"}
Modern touch slider for carousels and galleries.

[Documentation](https://swiperjs.com/){target="\_blank"}
:::

:::card{icon="i-lucide-smartphone" title="vite-plugin-pwa"}
Progressive Web App support powered by Workbox for Vite projects.

[Documentation](https://vite-pwa-org.netlify.app/){target="\_blank"}
:::
::
